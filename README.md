Project Overview : 

The Patient Vital Status Checker is developed using PHP that evaluates a patient's vital signs and determines
their health condition based on predefined rules.

Features :

  --- Input patient vital details
  --- Displays patient status (HIGH / LOW / NORMAL)
  --- Screenshot of output included

Key Concepts Implemented : 

  --- PHP Functions
  --- Conditional Statements(if-else)
